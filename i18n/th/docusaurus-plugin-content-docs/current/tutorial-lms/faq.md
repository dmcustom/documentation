# คำถามที่พบได้บ่อย
นี่คือคำถามที่พบบ่อยที่สุด พร้อมคำอธิบายว่าเหตุใดปัญหาเหล่านี้จึงเกิดขึ้น

## ทำไมจึงไม่มีเสียงระหว่างทดสอบใน Unity
- ตรวจสอบให้แน่ใจว่าใน Scene มี **Camera** วางอยู่ใกล้กับอวาตาร์ของคุณ
- ตรวจสอบว่าไม่ได้เปิดใช้งานปุ่ม **Mute Audio** ในหน้าต่าง Game
- ใช้เวอร์ชันล่าสุดของ [Gesture Manager](https://booth.pm/ja/items/3922472?srsltid=AfmBOorNhyQC8H3uU0CpaEoEBfJrnelkaT5MD-GRcsTh3hPsHCpz7B_i) เพื่อจำลองการโต้ตอบระหว่างการทดสอบ

## ทำไม DPS ไม่ทำให้เกิดเสียง?
DPS แบบ Legacy ไม่มี Avatar Contact Sender มาให้โดยค่าเริ่มต้น จึงไม่สามารถโต้ตอบกับ LMS ได้
ในกรณีนี้ คุณสามารถใช้ SPS Socket แทนได้

## ทำไมสิ่งของหรือวัตถุในโลกไม่ทำให้เกิดเสียง?
World props ไม่มี Avatar Contact Sender มาโดยค่าเริ่มต้น ซึ่งเป็นสิ่งจำเป็นสำหรับการโต้ตอบกับ LMS อย่างไรก็ตาม ตั้งแต่ **SDK 3.10.0** เป็นต้นมา สามารถเพิ่ม Contact component ลงในวัตถุภายในโลกได้แล้ว

## วิธีเพิ่ม Contact และทำให้โต้ตอบกับ LMS?
1. สร้าง GameObject เปล่า (Empty GameObject) จากนั้นเพิ่มคอมโพเนนต์ `VRC Contact Sender`
2. ปรับแต่ง Shape, Radius และค่าต่าง ๆ ตามต้องการ
3. ในหัวข้อ **Collision Tags** กด **Add** เลือก `Custom` และใส่ค่า: `LMS_Orf_Root`

## ทำไม LMS Object ของฉันขึ้น Missing Scripts?
โปรดตรวจสอบว่าคุณได้ติดตั้ง Avatar Tool ที่ถูกต้องสำหรับแพ็กเกจของคุณแล้ว ไม่ว่าจะเป็น Modular Avatar หรือ VRCFury.

## วิธีแก้ปัญหาขนาดผิดปกติ (ใหญ่เกินไป / เล็กเกินไป)
ปัญหานี้เกิดจาก Armature ของอวาตาร์ไม่ได้ตั้งค่า Scale เป็น (1,1,1) ทำให้ LMS ถูกวางขนาดใหญ่หรือเล็กเกินไป ซึ่งไม่ใช่ความผิดของคุณ
- วิธีแก้ไข: ปรับ Scale ของ GameObject ที่ชื่อ `LMS Contact` ให้ใหญ่หรือเล็กตามต้องการ
