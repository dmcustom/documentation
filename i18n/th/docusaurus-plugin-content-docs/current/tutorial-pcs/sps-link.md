# การเชื่อมเมนูกับ SPS

เนื่องจาก PCS ถูกออกแบบแตกต่างจาก SPS แม้ว่าทั้งสองระบบจะมีจุดประสงค์คล้ายกันในด้านการโต้ตอบแบบผู้ใหญ่ ความสะดวกในการใช้งานจึงเป็นสิ่งที่ผู้ใช้ส่วนใหญ่ให้ความสำคัญ — โดยเฉพาะการไม่ต้องสลับไปมาระหว่างเมนูบ่อย ๆ
แม้ว่า PCS จะไม่ได้ถูกสร้างมาให้ทำงานร่วมกับ SPS โดยตรง แต่เรายังคงสามารถเชื่อมโยงเมนูของทั้งสองระบบเข้าด้วยกันได้ในระดับหนึ่ง เพื่อให้การใช้งานสะดวกมากยิ่งขึ้น

## วิธีการเชื่อม
1. เลือก SPS Socket ใดก็ได้ที่คุณต้องการใช้ควบคุมเมนู PCS
2. เปิดใช้งานตัวเลือก `Enable Active Animation`
3. กด `Add Action +` แล้วเลือก `Set an FX Float`
4. กรอกค่า Parameter ของเมนู PCS ที่ต้องการลงในช่อง โดยมีรายการพารามิเตอร์ทั้งหมดแสดงไว้ด้านล่าง

## ผลลัพท์ที่ได้
- พารามิเตอร์ในเมนู **Sound & Location** ที่ถูกกำหนดให้กับ SPS Socket ที่เปิดใช้งาน จะถูกเปิดใช้งานโดยอัตโนมัติเช่นกัน
- ตัวอย่างเช่น หากเพิ่ม `pcs/select/pussy` ให้กับ SPS pussy socket เมื่อ Pussy Socket ถูกเปิดใช้งาน เสียงของ PCS สำหรับตำแหน่งดังกล่าวก็จะถูกเปิดใช้งานตามไปด้วย
