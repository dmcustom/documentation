## เกี่ยวกับระบบ
Penetration Contact System (PCS) คือระบบเสียงขั้นสูงที่ตรวจจับการสัมผัสและการโต้ตอบของอวาตาร์ พร้อมสร้างเสียงตอบสนองแบบไดนามิก โดยสามารถตรวจจับรูปแบบการสอดใส่ได้สูงสุด 5 รูปแบบ ซึ่งรวมถึงระดับความรุนแรงที่แตกต่างกัน จังหวะการเคลื่อนไหวเข้า–ออก การสไลด์ การดึงออก และอื่น ๆ

นอกจากนี้ PCS ยังมีระบบเสียงคราง และระบบแต้มค่าความเสียวในลักษณะคล้ายกลไกเกม Hentai เพื่อเพิ่มความสมจริงและความลื่นไหลของประสบการณ์การโต้ตอบ

## เงื่อนไขเพิ่มเติมสำหรับผู้จำหน่ายอวาตาร์
หัวข้อนี้จะครอบคลุมถึงผู้จัดจำหน่ายอวาตาร์แบบสำเร็จรูปที่มีการติดตั้งระบบ PCS ไว้แล้ว รวมถึงผู้ที่เปิดรับทำ Avatar Commission
และให้บริการติดตั้งระบบ PCS ให้แก่ลูกค้า

### กรณีเป็นผู้ขาย หรือ ผู้รับจ้าง
ผู้ขาย (Seller) หมายถึง บุคคลที่ทำการปรับแต่งหรือสร้างอวาตาร์แบบสำเร็จรูป เพื่อนำไปจำหน่ายต่อผ่านแพลตฟอร์มหรือเว็บไซต์ต่าง ๆ เช่น Gumroad, Jinxxy หรือแพลตฟอร์มอื่นที่มีลักษณะเดียวกัน

ผู้รับจ้าง (Commissioner / Service Provider) หมายถึง บุคคลที่ได้รับการว่าจ้างให้สร้างหรือปรับแต่งอวาตาร์ตามความต้องการเฉพาะของผู้ว่าจ้าง โดยดำเนินงานภายใต้ขอบเขตที่ตกลงกันไว้เป็นกรณีงานต่อกรณีงาน

คุณจำเป็นต้องมี `Commercial Use License` หากต้องการติดตั้งระบบ PCS ลงในอวาตาร์ที่นำไปจำหน่าย หรืออวาตาร์ที่ได้รับการว่าจ้างให้จัดทำ

ในการส่งออกแพ็คเกจอวาตาร์ คุณต้องยกเว้นไฟล์สคริปต์ของระบบ PCS ออกจากแพ็คเกจ เพื่อป้องกันการนำไฟล์ระบบไปแจกจ่ายต่อ

### กรณีเป็นผู้ซื้ออวาตาร์
ผู้ซื้ออวาตาร์ (Avatar Buyer) หมายถึง บุคคลที่ทำการซื้ออวาตาร์สำเร็จรูปผ่านแพลตฟอร์มหรือเว็บไซต์ต่าง ๆ เช่น Gumroad, Jinxxy หรือแพลตฟอร์มอื่นที่มีลักษณะเดียวกัน เพื่อใช้งานส่วนบุคคลและไม่ได้มีวัตถุประสงค์ในการนำอวาตาร์ดังกล่าวไปจำหน่ายต่อ แจกจ่ายต่อ

- คุณไม่จำเป็นต้องซื้อหรือครอบครองระบบ PCS เพื่อทำการซื้อหรือใช้งานอวาตาร์ที่มีการติดตั้ง PCS ไว้แล้ว
- คุณสามารถซื้อระบบ PCS แบบ `Personal Use License` มาเป็นของตนเองได้ หากต้องการอัปเดตเวอร์ชันของระบบ หรือทำการปรับแต่ง/แก้ไขการตั้งค่า PCS เพิ่มเติมให้กับอวาตาร์ที่คุณได้ซื้อมา

### กรณีเป็นผู้ว่าจ้าง
ผู้ว่าจ้าง (Client) หมายถึง บุคคลที่ว่าจ้างผู้รับจ้างให้ดำเนินการสร้าง ปรับแต่ง หรือติดตั้งระบบต่าง ๆ (รวมถึงระบบ PCS) ให้กับอวาตาร์ตามความต้องการเฉพาะของตนเอง โดยไม่มีวัตถุประสงค์ในการนำผลงานดังกล่าวไปจำหน่ายต่อ

- คุณจำเป็นต้องมีสิทธิ์ใช้งาน PCS แบบ `Personal Use License` และ ไม่จำเป็นต้องมี `Commercial Use License` เพื่อทำการว่าจ้าง
- เพื่อให้เข้าใจภาพรวมอย่างชัดเจน:
  - ผู้ว่าจ้าง ใช้ `Personal Use License` เท่านั้น
  - ผู้รับจ้าง ต้องใช้ `Commercial Use License` ในการดำเนินงาน
- อย่างไรก็ตาม หากคุณมีการกระทำใด ๆ ที่เปลี่ยนสถานะของตนเอง เช่น การนำอวาตาร์ที่ว่าจ้างจัดทำเสร็จแล้วไปจำหน่ายต่อเพื่อแสวงหาผลกำไร คุณจะถือว่ามีสถานะเป็น ผู้ขาย (Seller) โดยทันที และต้องปฏิบัติตามข้อกำหนดและเงื่อนไขในส่วนของผู้ขายแยกต่างหาก
