# วิธีการใช้งาน

เนื่องจาก PCS ใช้ระบบ Avatar Contact ในการตรวจจับ
ดังนั้นคุณจำเป็นต้องเปิดใช้งานสิทธิ์การสัมผัสระหว่างผู้เล่น (player-to-player contact permissions) ในเมนูตั้งค่าเกม มิฉะนั้น PCS จะไม่สามารถทำงานได้!

โดยค่าเริ่มต้น ระบบสามารถตรวจจับการสอดใส่หรือการโต้ตอบกับ SPS Plugs และนิ้วของอวาตาร์ได้ หากต้องการให้ PCS ตรวจจับกับสิ่งของอื่นนอกเหนือจากนี้ โปรดอ่านเพิ่มเติมได้ที่ [How to Add Contact](./faq#how-to-add-a-contact-and-make-it-interact-with-pcs)

## 1. การตั้งค่า Avatar Interaction
นี่คือการตั้งค่าภายในเกมที่จำเป็นต้องทำเพื่อเปิดใช้งานระบบ Avatar Contact
<img src={useBaseUrl('/img/pcs-contact-setting.png')} width="540" />
- **Pause Avatar Interaction** > `OFF`
- **Avatar Allowed to Interact** > `Friends, Everyone`
- **Avatar Self Interact** > `ON`

## 2. เปิดใช้งาน PCS
1. เข้าไปที่เมนู PCS แล้วสลับเมนู **ON/OFF** ให้เป็น `ON`
2. เลือกเสียงที่ต้องการใช้ในเมนู **Sound & Location**
   - สามารถเลือกเสียงหรือตำแหน่งได้เพียงอย่างใดอย่างหนึ่งในแต่ละครั้ง
   - เสียงและตำแหน่งจะเชื่อมโยงกัน เว้นแต่คุณจะใช้เมนู [Custom Position Menu](./avatar-setup#custom-position) เพื่อเปลี่ยนตำแหน่งแต่ยังคงใช้เสียงเดิม
3. เมื่อมีการสัมผัสจากนิ้ว หรือ SPS Plug ระบบจะสร้างเสียงตามที่ตั้งค่าไว้

:::tip คำแนะนำ
หากคุณอยู่ในบทบาทฝ่ายที่เป็นผู้เคลื่อนไหว คุณสามารถใช้โหมด **Self-Service** เพื่อสร้างเสียงได้ง่ายขึ้น
เมนูนี้ช่วยให้คุณสร้างเสียงตามการเคลื่อนไหวของตัวเองได้ แม้จะไม่ได้มีการสัมผัสกับผู้อื่นก็ตาม
:::
